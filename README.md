# interactive-gltf-specs
This project intends to specify an extension of the GlTF format making it interactive.

**Language:** Specification and proposal documents under `specifications/` and `proposals/` are written in **English**.

If **`UMI3D-SDK-version-2.9`** is in the workspace (often as a second root, e.g. with `docs/` and `UMI3D-SDK/`), it is **read-only** reference material for UMI3D; do not change that tree — only evolve the interactive-gltf spec in this repository.
